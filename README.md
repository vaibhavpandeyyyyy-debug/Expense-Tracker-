# 💸 Monthly Expense Tracker

Dark-themed Flutter app to track monthly expenses with categories, charts, and budget limits.

## Features
- ✅ Add & delete expenses
- 📁 8 categories (Food, Rent, Transport, Shopping, Health, etc.)
- 📊 Pie chart + category-wise summary
- 💰 Monthly budget limits with alerts
- 🌙 Dark Purple theme
- 💾 Data saved locally on phone

---

## How to Get APK (Free, No Setup Needed)

### Step 1: Upload to GitHub
1. Go to [github.com](https://github.com) → Sign up / Login
2. Click **New Repository** → Name it `expense-tracker` → Click **Create**
3. Upload ALL files from this folder (drag & drop)

### Step 2: GitHub Builds APK Automatically
1. Go to your repo → Click **Actions** tab
2. You'll see a workflow running — wait ~5 minutes
3. Once done, click the workflow → scroll down to **Artifacts**
4. Download **expense-tracker-apk**
5. Extract the zip → install the `.apk` on your Android phone!

### Enable Unknown Sources (One Time)
On your Android phone:
- Settings → Security → Enable **Install Unknown Apps**
